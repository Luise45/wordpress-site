# Wordpress Travel Blog 


---

## Projektübersicht
Dieses Projekt ist wurde erstellt um den Umgang mit dem CMS Wordpress zu üben. 
Der Blog wurde mit WordPress aufgebaut und angepasst. Wegen der kosten, ist die Seite nicht öffentlich gehostet, sondern wird nur über Screenshots gezeigt.

---

## Verwendete Technologien & Tools
Für die Erstellung der Website wurden folgende Tools und Plugins verwendet:
- **WordPress** CMS
- **Elementor** Page Builder 
- **Astra Theme** leichtes und flexibles Theme
- **Plugins**  z. B. für Formulare und zusätzliche Funktionen 
- **Forms (Kontaktformulare)** 
- **Widgets** für zusätzliche Inhalte und Layout-Elemente wie z.B. Google Maps
- **Pages & Posts** Seiten & Blog-Beiträge
